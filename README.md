# Menu Driven Singly Linked List in C

A simple *menu-driven C program* that implements a *singly linked list*.  
It allows users to create a list, insert a node at the beginning, display nodes, and delete the last node.

---

##  Features
- Create a linked list  
- Insert at beginning  
- Display all nodes  
- Delete at end  
- Uses dynamic memory allocation  



##  Menu
1. Create List  
2. Insert at Beginning  
3. Display List  
4. Delete at End  
5. Exit  



##  Concepts
- Structures & Pointers  
- `malloc()` and `free()`  
- Singly Linked List  
- Switch-case menu
